517: [Heading 3] 4.2.1. AI Agent Implementation
518: [Normal] AI Agent được xây dựng bằng LangChain với kiến trúc Tool Calling. Agent chính (VenueAgent) sử dụng LLM để phân tích ý định người dùng và quyết định gọi tool phù hợp.
519: [Normal] Cấu trúc Agent:
520: [List Bullet] VenueAgent: Agent chính, xử lý chat message.
521: [List Bullet] IntentRouter: Phân loại intent nhanh (embedding-based) cho các câu hỏi đơn giản.
522: [List Bullet] SimpleVenueAgent: Fallback agent khi LLM không khả dụng.
523: [List Bullet] Tools: Các hàm xử lý nghiệp vụ (book_court, order_food, call_staff, query_knowledge).
524: [Normal] Luồng xử lý:
525: [Normal] 1. Nhận message từ người dùng.
526: [Normal] 2. IntentRouter kiểm tra intent đơn giản (greeting, goodbye).
527: [Normal] 3. Nếu phức tạp, gọi LLM AgentExecutor.
528: [Normal] 4. LLM phân tích và gọi tool phù hợp.
529: [Normal] 5. Tool thực thi, trả kết quả về LLM.
530: [Normal] 6. LLM tổng hợp phản hồi cuối cùng.
